# Read Me First
A simple Springboot demo app using [langchain4j](https://docs.langchain4j.dev/)

# How to run

1. git clone this repo
2. create a token for GitHub. This demo uses GPT_4_O_MINI from GitHub Copilot.
3. set env var GITHUB_TOKEN=<token>
4. open the Gradle project in your IDE
5. run the Springboot app
